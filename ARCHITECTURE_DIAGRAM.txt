KEY DECISION POINTS
═══════════════════════════════════════════════════════════════════════════════

1. CHANNEL FOUND?
   Yes → confidence="auto"
   No  → confidence="unmatched", skip reporting for this team

2. EMAIL ALREADY PROCESSED?
   Yes → skip (deduplicate)
   No  → mark as processed, continue

3. COMPLETE PAIRING DATA?
   Yes → use parsed data
   No  → try LLM fallback
   Still No → skip email

4. OUR SCHOOL'S TEAM?
   Yes → process
   No  → skip

5. OPPONENT SIDE KNOWN?
   Yes → look up caselist
   No  → skip caselist lookup, note "FLIP"

6. CASELIST FOUND?
   Yes → LLM summarize
   No  → show "_No caselist data found._"

7. JUDGE NAME VALID?
   Yes → look up paradigm + Notion
   No  → skip judge (filter out signatures, dashes)
